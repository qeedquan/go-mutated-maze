Godspeed You! Mutated Maze
==========================

Entry in PyWeek #13 <http://www.pyweek.org/13/>
Team: SgtHagfish13
Members: Nick Sonneveld


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/


RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py


HOW TO PLAY THE GAME:

Press <SPACE> to play.

Collect the key to open the door.  Collect friendly mobs to bring to the exit.
Avoid the colourless ghosts.

Move your character around with the arrow keys.

Press the left mouse button to use your weak mutator. It can mutate the maze
but not move around mobs (unlike the more powerful fog).
